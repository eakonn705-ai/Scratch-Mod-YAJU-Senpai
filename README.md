# NIGHT-MODとは？
NIGHT-MODとは、MITメディアラボが開発した子供向けビジュアルプログラミング言語「Scratch」の実行専用MODです。
eakonn705によりClaudeSonnetを用いたバイブコーディングで5日で開発されました。
ウェブサイトから手軽に使うことができます。

## 使用方法
下記のリンクにアクセスしてください。詳細な使用方法は、サイト内のUSER GUIDEを読んでください。
アカウント登録などは不要です。
稀に、GeminiAPIの関係上、混雑によりScratch Copilotが使用できない場合があります。その場合は数分ほど時間を空けてからお試しください。

https://eakonn705-ai.github.io/Night-Mod/

## 主な独自機能
1. fpsスライダー：fpsを30から120までスライダーを用いて自在に変更できます。
2. 新UI：変数やリストなどの見た目がスクラッチとは異なります。
3. Scratch Copilot：GeminiAPIによるAIアシスタントです。モデルにはGemini 3.1 Flash Liteを使用しています。
   
   スクラッチについての質問（ブロックの種類など）について答えることができます。
   sb3ファイルを熟読した状態なら、解析してバグを検出したり、新提案をしたりもできます。稀に混雑して使えません。その場合はもう一度送信してください。

## 利用規約
### Scratch について
Scratchは scratch.mit.edu で提供されるMITのプロジェクトです。Scratchの名称・ロゴはMITメディアラボの商標です
### プライバシー
Scratch Copilotに入力した内容はGAS経由でGemini APIに送信されます。個人情報や機密情報は入力しないでください。
送信データはGoogleのプライバシーポリシーに従って処理されます。十分お気を付けください。
